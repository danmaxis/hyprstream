Tap to toggle the default microphone mute via PipeWire.

The icon mirrors live state at a glance: green mic when live, red mic with a slash when muted. State is polled every 500 ms while this action is on screen, then polling stops to save CPU. Mute changes from the keyboard, the system tray, or any other app appear here within half a second.

The action targets @DEFAULT_AUDIO_SOURCE@ — whichever input device PipeWire considers the default. Switch defaults in pavucontrol or wpctl set-default … if you have multiple mics.
Requires: wpctl (Arch: sudo pacman -S wireplumber) — already installed if you're on a current Hyprland setup.
