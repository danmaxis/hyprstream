Mirrors the current MPRIS track into an OBS text source (and optionally an image source for album art) over obs-websocket v5 — the overlay updates itself on every track change. Placeholders: {artist}, {title}, {album}. The key's OBS badge turns green when connected; press the key to force a re-push.

Enable the obs-websocket server in OBS: Tools → WebSocket Server Settings. For album art, add an Image source in OBS and put its name above. Text now-playing works with any OBS install (native or Flatpak) — it's sent over the socket.

Flatpak OBS + album art: the sandbox can't read arbitrary host files, so grant it the art folder once:
flatpak override --user --filesystem=xdg-run/hyprstream-media:ro com.obsproject.Studio
(restart OBS afterward). Native/manual OBS needs no override. To use a different folder, set it above and grant that path instead.

Requires: playerctl (pacman -S playerctl) and OBS 28+ with obs-websocket v5
