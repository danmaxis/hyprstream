Drives whichever MPRIS player is active (Spotify, mpv, browsers, VLC, etc.) via playerctl. Play/Pause is dynamic — its icon flips between ▶ and ❚❚ to mirror the current playback status. Next and Previous are static. Status polls at ~1Hz while the action is on screen.

Album art reads the focused player's mpris:artUrl and composites it under the play/pause glyph. Local file:// art (mpv, Rhythmbox, browser players that cache locally) always loads. To pull HTTPS art from streaming services, flip the second toggle — disabled by default so the plugin doesn't make outbound network calls without consent.

Requires: playerctl (pacman -S playerctl)
