Each rule watches a metric and, on crossing its threshold, drives OBS: fill a text source, show/hide a source, and/or swap an image file — any combination; leave a reaction's fields blank to skip it. Text templates support {value}, {label}, {unit}. Set a clear value looser than the threshold for a dead-band that stops flapping (e.g. CPU fire ≥90, clear <80). Overlays are restored when the key is removed.

Requires OBS 28+ with obs-websocket v5.
